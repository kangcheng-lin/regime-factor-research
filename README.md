# Regime Factor Research

This project aims to rebuild and extend regime-based factor investing models from academic literature.

## Data Sources

- S&P 500 ETF data: Yahoo Finance  
- Fama-French Factors (MKT, SMB, HML, MOM): Kenneth French Data Library  
  https://mba.tuck.dartmouth.edu/pages/faculty/ken.french/data_library.html  
- Quality Minus Junk (QMJ): AQR Data Library  
  https://www.econ.yale.edu/~af227/data_library.htm
